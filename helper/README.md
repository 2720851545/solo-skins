B3log Solo 皮肤工具
==================================================

PSD
------------------------------------------
存放皮肤中用到的 psd 文件

skin-preview
------------------------------------------
存放每个皮肤的预览图片。命名规则为： skinName[-pathname].png

tools
------------------------------------------
皮肤辅助工具（Node.js）
* compress.js
  对 js, css 进行压缩后在当前目生成 XXX.min.js/css 文件
* get-static.js
  提取皮肤中的 js, css, image 等静态资源到 b3log-solo-skins/helper/tools/static 目录下
* update-version.js
  批量更新 skin.properties 文件中的 forSolo 版本号
